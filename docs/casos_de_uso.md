# Casos de Uso Principales

## Administrador

### CU-01. Gestionar usuarios

Puede crear, editar, desactivar y consultar usuarios del sistema.

### CU-02. Gestionar materias primas

Puede crear, editar, eliminar logicamente y consultar materias primas.

### CU-03. Gestionar formulaciones

Puede crear, editar, consultar y desactivar formulaciones.

### CU-04. Revisar alertas

Puede consultar todas las alertas y cambiar su estado.

## Profesor

### CU-05. Registrar formulaciones

Puede crear formulaciones y sus componentes.

### CU-06. Editar formulaciones

Puede actualizar formulaciones existentes segun autorizacion institucional.

### CU-07. Consultar materias primas

Puede revisar disponibilidad, vencimientos y datos tecnicos.

### CU-08. Revisar alertas

Puede consultar alertas de vencimiento y stock.

## Estudiante

### CU-09. Consultar formulaciones

Puede revisar formulaciones con fines academicos.

### CU-10. Consultar materias primas

Puede consultar materias primas y sus datos visibles.

## Casos automaticos del sistema

### CU-11. Generar alerta por vencimiento proximo

El sistema debe evaluar diariamente si una materia prima vence dentro del umbral definido.

### CU-12. Generar alerta por caducidad

El sistema debe marcar la materia prima como vencida y generar alerta critica.

### CU-13. Generar alerta por stock bajo

El sistema debe generar alerta cuando el stock actual sea menor o igual al stock minimo.

### CU-14. Generar alerta por falta de stock

El sistema debe generar alerta critica cuando el stock actual sea cero.
